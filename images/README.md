## Want to change captions or add/remove photos?
Edit the `GALLERY` list at the top of [`../js/main.js`](../js/main.js) — it's plain
English. Each line points at a file name here and its caption/category. Add a new
photo to this folder, add a matching line to `GALLERY`, done.

## Tips
- Square photos (~1000×1000) look best in the portfolio grid.
- Keep files under ~500 KB where you can, so the site loads fast.
- File names are case-sensitive on GitHub — keep them lowercase.
- A missing file shows a striped placeholder instead of breaking the page.
